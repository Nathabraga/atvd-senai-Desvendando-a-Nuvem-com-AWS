# 🌩️ AWS GERAL

## 📌 O que é a AWS?

A AWS (Amazon Web Services) é uma plataforma de computação em nuvem da Amazon.

Ela permite que pessoas e empresas utilizem serviços como servidores, armazenamento e banco de dados pela internet, sem precisar comprar computadores físicos.

---

## 🌍 Infraestrutura Global

A AWS possui servidores espalhados pelo mundo.

### 🗺️ Regiões

As regiões são locais onde existem data centers da AWS.

* Estão em vários países
* Funcionam de forma independente
* Ajudam a deixar os serviços mais rápidos

---

### 🏢 Zonas de Disponibilidade

Dentro das regiões existem as Zonas de Disponibilidade.

* São vários data centers
* Funcionam separados
* Se um falhar, outro continua funcionando

Isso garante mais segurança e estabilidade.

---

## 💰 Economia da Nuvem

### 💳 Pay-as-you-go

A AWS usa o modelo Pay-as-you-go, ou seja:

* Você paga só pelo que usar
* Não precisa pagar antes
* Pode usar mais ou menos quando quiser

---

### 📈 Economia de escala

A AWS consegue preços mais baratos porque atende muitas pessoas.

* Compra muitos equipamentos
* Reduz custos
* Atende muitos clientes

Assim, ela ganha dinheiro com o grande número de usuários.

---

## 🚀 Conclusão

A AWS é uma forma prática, segura e econômica de usar tecnologia pela internet.
